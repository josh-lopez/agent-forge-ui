# Contributing to agent-forge-ui

Thank you for your interest in contributing to agent-forge-ui!

## How to Propose Changes

### Issues
- Open an issue to describe a bug, feature request, or improvement.
- Provide clear context and expected behavior.

### Pull Requests
- Create a pull request to propose code changes.
- Link your PR to the related issue if applicable.
- All PRs require review and approval before merging.

## Branch Naming

Use descriptive branch names that reflect the work:
- Feature branches: `feature/short-description`
- Bug fixes: `fix/short-description`
- Documentation: `docs/short-description`

Example: `feature/add-dark-mode` or `fix/button-alignment`

## Code Review

- All pull requests must be reviewed by at least one maintainer.
- Address feedback and update your PR as needed.
- Once approved, your changes will be merged.
